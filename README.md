# Human-Assistive-Glove-project
Final Year Project -Smart Glove for Sign Language Translation and Gesture Controlled Home Automation

## ğŸ“Œ Project Overview
This project is designed to help speech-impaired and differently-abled people communicate with normal people using hand gestures. A wearable smart glove is used to detect hand movements and translate them into meaningful text and speech. The same glove can also be used to control home appliances like fan and light using hand gestures.
The smart glove is integrated with sensors and microcontrollers to detect motion and identify different gesture patterns.
